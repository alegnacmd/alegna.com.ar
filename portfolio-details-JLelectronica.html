Diseño de pagina web desde cero

Creé una página web para Electrónica JL utilizando Shopify.

Como la página funcionaría principalmente como catálogo, no fue necesario integrar una pasarela de pago. Implementé un formulario mediante el cual el usuario enviaba su carrito para completar la compra. Este proceso incluía una redirección que enviaba instrucciones al cliente por mail, mejorando la confianza y claridad del proceso. Diseñé banners, modifiqué tipografías del template, realicé la carga masiva de productos (nombre, descripción, fotos), agregué filtros personalizados y modifiqué header y footer incorporando íconos y enlaces a redes.

Shopify (plan básico), Photoshop, Illustrator y Google Sheets para la carga masiva.

Se armó una estructura que funciona tanto de catálogo visual como de carrito, facilitando así el trabajo diario de carga y organización de productos, esto agilizó la selección y compra tanto para el usuario como para el cliente.
